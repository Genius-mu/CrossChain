# Web3 Chain Bridge

A secure, fast, and user-friendly cross-chain bridge for transferring assets between Ethereum, Polygon, and Arbitrum networks.

## ðŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**

   ```bash
   https://github.com/Genius-mu/CrossChain.git
   cd web3-chain
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**

   Navigate to [http://localhost:5173](http://localhost:5173) to view the application.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

### Run Linting

```bash
npm run lint
```

## ðŸ“– Usage

1. **Connect Wallet**: Link your MetaMask or WalletConnect wallet
2. **Select Networks**: Choose source and destination blockchains
3. **Choose Token**: Select from supported tokens (ETH, USDC, USDT, DAI)
4. **Enter Amount**: Input the amount you want to bridge
5. **Review Details**: Check estimated time, fees, and security status
6. **Confirm Transaction**: Approve the transaction in your wallet
7. **Wait for Completion**: Transfer typically completes in under 2 minutes
